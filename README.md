                                    UsFer framework
                                A useful Fernet framework.

UsFer is a tool built in python used to encrypt strings and save them to files, built using the Fernet library.

Things to look out for:
  ~ Strange bug with file names.
  ~ Recommended to generate a new key for first time use.
